06 / 10
Technical stack

Zero framework. Zero build step.

Frontend
Vanilla JS + Canvas 2D
One game.js. No React, no bundler. Pixelify Sans + VT323 + Outfit for the three-tier type hierarchy. Web Audio API for procedural sound.
Server
Node.js serve.js
230 lines. HTTP static serving AND WebSocket relay on one port. No Express. The relay is a Map of room codes → display + controller sockets.
Hosting
Render free tier
Auto-deploys on GitHub push. Custom /health endpoint kept warm by UptimeRobot. Has stayed live through every in-class session without a cold start.
Persistence
Upstash Redis
Key-value database on AWS North California. Leaderboard stored as JSON at key sfr:scores. REST API only — no SDK, no connection pool, survives any restart.
